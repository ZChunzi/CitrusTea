## 使用方法

1.拉取项目：

```git clone https://github.com/ZChunzi/CitrusTea.git```

2.我猜你肯定安装了NodeJs，运行以下指令：

```
cd CitrusTea
npm install
```

3.运行，输入以下指令运行：

```npm run dev```

或者

```node server.js```

然后访问

```http://localhost:3000```

出现这个就成功了

![image](https://github.com/ZChunzi/CitrusTea/assets/63871424/f16af6d1-31f4-4c41-9723-85174af01ee2)

编译：

```npm run build```

编译后的文件将出现在dist文件夹里
