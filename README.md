# CitrusTea

## 使用方法

1.拉取项目：

```git clone https://github.com/ZChunzi/CitrusTea.git```

2.NodeJs，运行以下指令：

```
cd CitrusTea
npm install
```

3.运行，输入以下指令运行：

```npm run dev```

或者

```node server.js```

然后访问

```http://localhost:3000```
